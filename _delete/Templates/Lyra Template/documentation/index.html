INTRODUCTION

Hello and thank you for using Lyra SaaS App Landing Page Template. This landing page was designed to help you establish an awesome online presence for your SaaS application and convert visitors into buying customers. It features the following versions: Vertical Registration, Horizontal Registration, Youtube Background, Video In Header, Background Slider, Hero Slider, Countdown Timer and Coming Soon.

Built with HTML5, CSS3, PHP and JavaScript it will help you achieve your online marketing goals in a beautiful and efficient manner.

Frameworks and toolkits used: Bootstrap v3.3.7, Themify Icons, Animate.css v3.5.1, jQuery v2.2.4, jQuery Easing v1.3, Validator.js v0.10.2, Magnific Popup v1.1.0, Swiper v3.4.0, Morphtext v2.4.4, jQuery YTPlayer and jQuery Countdown v2.2.0

You can edit the template's Html, Css, Php, Js files directly with Notepad++ or with your preferred code editor.

For graphics editing you can use Adobe Photoshop or any other graphics software of your choice.

Background Slider Images

To change the images that slide in the header background of the Background Slider version, you need to:
- Open styles.css from the css folder inside the template
- Find the line: .header .swiper-container-header .swiper-slide.first .slider-background
- And replace: http://placehold.it/1920x1080/3a3a3a/fff/ with the url to your image e.g. images/background-slide-1.jpg
- And repeat for each slide: first, second and third

Youtube Background

To change the video in the background of the Youtube Background version, you need to:
- Open index.html inside the template folder
- Find the line 

<a id="bgndVideo" class="player" data-property="{videoURL:'https://www.youtube.com/watch?v=_smyOW8JKSw',containment:'.video-section', quality:'highres', autoPlay:true, showControls: false, startAt:0, stopAt: 15, mute:true, opacity: 1}"></a>

- Go to your Youtube video copy the link from the browser's address bar and paste it in the line above to replace the default link which looks like this https://www.youtube.com/watch?v=_smyOW8JKSw inside index.html

Video In Header

To change the video found in the header of the Video In Header version, you need to:
- Open index.html inside the template folder
- Find the line <iframe src="https://www.youtube.com/embed/e6TWX-6s13s" frameborder="0" allowfullscreen></iframe>
- Go to your Youtube video, click on Embed, copy just the link that looks like this e.g. https://www.youtube.com/watch?v=_smyOW8JKSw (without any other parameters) and replace it in the above mentioned line inside index.html

ANIMATIONS

Animate.css

Animate.css is used to animate the loading sequence and the contact form success message.
You can use it to animate any item by adding two classes to any html item: the name of the animation and the animated class.
You can find animations names and their preview on Animate.css web page.

Rotating Text

You will see in almost all versions of the template, a word that changes to others by rotation in the Header section. To change these words you need to:
- Open index.html from the template folder
- Find the line: <h1>SaaS App Landing Page <span id="js-rotating">Template, Business, Marketing, Sales</span></h1>
- And change: Template, Business, Marketing, Sales to the words that you need
- To change the background color, open styles.css from the css folder
- Find the lines: 

header .header-content h1 #js-rotating .animated {
	background-color: #298aff;
}

- And replace background-color: #298aff; with the color you need

ICONS

Social Media Icons

To change social media icons in the Footer section of the template, you need to:
- Open index.html from the template folder
- Find the lines: 

<h4>Social Networks</h4>
<a href="#"><span class="social-icons ti-facebook"></span></a>
<a href="#"><span class="social-icons ti-twitter"></span></a>
<a href="#"><span class="social-icons ti-google"></span></a>
<a href="#"><span class="social-icons ti-linkedin"></span></a>
<a href="#"><span class="social-icons ti-instagram"></span></a>

- And replace ti-facebook, ti-twitter with the classes name that you need from https://themify.me/themify-icons in the Brand Icons section
- Also don't forget to replace href="#" with the link to your social page or account

Social Media Tip

If you want to control the way the posts look like when you share the landing page on social media (Facebook, Google+ and LinkedIn especially) don't forget to update the OG Meta Tags (OpenGraph) in the Head section of the index.html file. 

<meta property="og:site_name" content="" /> <!-- website name -->
<meta property="og:site" content="" /> <!-- website link -->
<meta property="og:title" content=""/> <!-- title shown in the actual shared post -->
<meta property="og:description" content="" /> <!-- description shown in the actual shared post -->
<meta property="og:image" content="" /> <!-- image link, make sure it's jpg -->
<meta property="og:url" content="" /> <!-- where do you want your post to link to -->
<meta property="og:type" content="article" />
You can check out this article courtesy of Engage Interactive

COLORS

All the color settings are found in styles.css inside css folder of the template.
You can change the primary color throughout the entire template in just a few steps:
- Open styles.css from css folder with Notepad++
- Find the primary color from one of the buttons, for example like in the following lines: 

.button {
	border: 2px solid #298aff;
	color: #298aff;
	border-radius: 25px;
	padding: 14px 45px 13px 45px;
	font: 600 14px/1em "Raleway", sans-serif;
	display: inline-block;
	margin-top: 15px;
	-o-transition: all .3s ease;
    -webkit-transition: all .3s ease;
    -moz-transition: all .3s ease;
    transition: all .3s ease;
}

- In our case it's: #298aff
- Go to Search menu in Notepad++ then Replace...
- In Find what dialog write: #298aff
- In Replace with dialog write the code of your new color e.g. #0fd777
- Now click on Replace All and you are done. Everywhere you had blue now you have green: buttons, features, pricing table...

CONTACT FORM

To receive emails with data from your visitors simply change the line $EmailTo = "yourname@domain.com"; to your email address in the php folder in the following files:

  • contactform-process.php, registrationform-process.php and newsletterform-process.php

Depending on the version of the template you will find one or more of the files above.

The script only runs online (on a liver server with PHP support) so the forms will not work while the landing page is offline on your hard-drive

If you want to change input fields names, you will need to update them accordingly in the following files:

  • index.html, scripts.js and the php files from the php folder depending on the template version
